# Grupo TB - Checkpoint 2

1. **Criar a virtualenv**:

   ```bash
   python3 -m venv venv
   ```

2. **Ativar o ambiente**:

   - macOS / Linux:

     ```bash
     source venv/bin/activate
     ```
   - Windows (PowerShell):

     ```powershell
     .\venv\Scripts\Activate.ps1
     ```

3. **Instalar as dependências**:

   ```bash
   pip install -r requirements.txt
   ```
